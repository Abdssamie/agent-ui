# Agent Instructions

## Design Guidelines

### Text Colors

**Do NOT use `text-muted-foreground`** - it has very low contrast (#71717A) against our dark backgrounds and makes text barely readable.

**Use instead:**
- `text-muted` (#A1A1AA) - for secondary/descriptive text
- `text-secondary` (#f5f5f5) - for readable light text
- `text-foreground` (#FAFAFA) - for primary text

### Color Reference
- `text-muted-foreground`: #71717A ❌ Too dark
- `text-muted`: #A1A1AA ✓ Better contrast
- `text-secondary`: #f5f5f5 ✓ Good contrast
- `text-foreground`: #FAFAFA ✓ Best contrast
